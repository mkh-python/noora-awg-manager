# Noora AWG Manager

Telegram management bot and installer for AmneziaWG.

## Install

Run:

    sudo bash -c "$(wget -qO- https://raw.githubusercontent.com/mkh-python/noora-awg-manager/main/install.sh)"

Installer asks for:

- Telegram BOT_TOKEN
- Telegram numeric OWNER_ID

After install, open your bot and send:

    /start

## Features

- Add AmneziaWG users
- QR and config delivery
- Traffic quota
- Expiry date
- Admin management
- Backup channel
- Full server backup
- Automatic backup schedule
- Domain and SSL management
- Web panel URL management

## Important

Do not run the installer on a server that is already configured unless you want to reinstall the bot files.
